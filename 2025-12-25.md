# 그림 리퀘스트 사이트 만들기

- 반응형으로 제작한다
- 안내문을 만든다
- 메세지 작성칸을 만든다
- 보내기 버튼을 만든다
- 보내기 버튼을 누르면 해당 메세지가 내 이메일로 도착하도록 만든다
- 디자인 테마: 도트

## 시간이 되면 만들어 볼 것

- 어떤 메세지를 보냈는지 확인 가능한 창
- 메세지를 보내면 메세지를 보냈습니다!라고 하는 모달창 띄워보기
- 버튼에 커서가 올라가면 색이 변하도록 만들어보기
- 무슨 메세지를 보냈는지 전체 타임라인 만들기
- 주의사항 창으로 쓸 이미지 그려보기
- 배경 이미지 그려보기
- 버튼 이미지 그려보기
