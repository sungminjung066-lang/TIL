## 개념 정리

#### TanStack Query

- 서버(API)에서 받아온 데이터를 관리해주는 라이브러리
- [링크](https://tanstack.com/query/latest/docs/framework/react/installation)

#### Query Key

- Query를 구분하기 위한 고유 이름표

#### useQuery

- 서버 데이터를 가져와서 캐싱하고 상태를 관리하는 TanStack Query Hook

#### QueryClient

- 모든 query 캐시와 설정을 중앙에서 관리하는 TanStack Query의 관리자 객체

#### staleTime

- 서버 데이터를 다시 요청하기 전까지 캐시 데이터를 유효하다고 판단하는 시간

#### Optimistic Update

- 서버 응답을 기다리지 않고, 성공했다고 가정하고 먼저 화면을 업데이트하는 방식
- ![이미지](<img/화면 캡처 2026-01-28.jpg>)

#### Mutation

- 서버 데이터를 변경하는 요청을 처리하는 TanStack Query 기능

#### Zustand

- React에서 전역 상태(Global State)를 간단하게 관리하는 라이브러리

- [링크](https://zustand.docs.pmnd.rs/getting-started/introduction)
