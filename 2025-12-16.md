### nodejs란?

프로그램 / 프로세스

프로그램 - 디스크 상에 설치된 코드의 묶음
프로세스??

비실행과 실행 중입니다
소프트웨어/실행 중인 프로그램
프로그램은 어플이고, 그어플을 실행하는 건 프로세스

우리가 지금 하려고하는 건 뭐죠?
-> 서버를 개발하자!
서버(프로그램)를 개발했다면, 이제 서버를 동작하게하려면?? 서버를 "시작!!"

서버가 실행(기동)되면 서버 프로세스가 생기는 것

Q. 브라우저도 프로세스임??
브라우저도 실행하면 프로세스임
브라우저를 통해 index.html, app.js를 실행하고 있음

브라우저 자체만 놓고보면 프로그램
-> 일련의 소스코드들로 개발되어 있음

우리가 앞으로 서버를 JS로 개발했다면, 하려면
JS만 가지고 별도의 서버 프로세스가 실행되어야 함

Q. 서버를 브라우저로 실행하실건가요?
브라우저(Browser) 내에서 서버를 실행???(X)

결론.
JS만 가지고 별도의 서버 프로세스가 실행되어야 함

그 말은 브라우저에서 실행할 수 있었던 JavaScript 런타임을
브라우저 밖으로 꺼내야 함

- JavaScript 런타임??
  JavaScript 코드를 실행할 수 있는 실행 환경이 구성된 프로그램

JS런타임은 기본적으로 각각의 브라우저 회사별 브라우저 소프트웨어 내부에 내장되어 있음
-> 브라우저 안에 있는 JS 런타임 프로그램 코드만 브라우저 밖으로 꺼내서
독립적으로 사용할 수 있게 만들어진 라이브러리가 Node.js

Node.js???

- JavaScript 실행할 수 있는 JS 런타임입니다
  -> 그러다보니 서버 개발에도 활용할 수 있음(주로 서버개발에 쓰기 때문에)
