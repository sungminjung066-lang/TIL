### jQery

- jQuery는 브라우저마다 다른 DOM, 이벤트, Ajax 구현을 하나의 API로 통합하여 웹 개발자가 크로스 브라우징 문제에서 해방되도록 만든 JavaScript 라이브러리

### Sass

- Sass 란 Syntatically Awesome Style Sheets의 줄임말로, CSS 코드를 보다 체계적으로 관리하고 유지보수성을 높이기 위해 고안된 확장 문법의 역할을 수행하는 CSS 전처리기(Pre-Processor)이다. 이를 통해 개발자는 Sass만의 추가 문법을 통해 CSS 코드를 보다 편하게 관리할 수 있음
